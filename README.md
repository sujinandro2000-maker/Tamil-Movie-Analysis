# 🎬 Tamil Movie Analysis (2011–2019)

## 📖 Project Overview
This project explores Tamil movie data from 2011 to 2019 using Python. The analysis focuses on understanding trends in movie ratings, genres, directors, actors, and audience votes through Exploratory Data Analysis (EDA).

## 🎯 Objectives
- Analyze movie trends over the years.
- Identify the most popular genres.
- Compare ratings across genres and years.
- Find the most successful directors and actors.
- Visualize insights using charts and graphs.

## 📂 Dataset
The dataset contains information about Tamil movies released between **2011 and 2019**.

### Columns
- MovieName
- Genre
- Rating
- Director
- Actor
- PeopleVote
- Year
- Hero_Rating
- movie_rating
- content_rating
